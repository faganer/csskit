CSSkit

CSSkit 是一组轻量级、响应式纯CSS基础模块，适用于任何Web项目。

开始使用：

<!-- 非自适应 -->
<link rel="stylesheet" href="../dist/css/csskit.min.css">

<!-- 自适应 -->
<link rel="stylesheet" href="../dist/css/csskit.min.css">
<link rel="stylesheet" href="../dist/css/grids-responsive.min.css">

<!-- 自适应 -->
<link rel="stylesheet" href="../dist/css/csskit.all.min.css">

<!-- 引入 jQuery -->
<script src="https://cdn.jsdelivr.net/npm/jquery@1.12.4/dist/jquery.min.js"></script>

<!-- 引入 JavaScript -->
<script src="../dist/js/csskit.min.js"></script>

Flex

用法

要应用flexbox布局模型，请使用以下类之一。默认情况下，所有flex项都与左侧对齐，宽度与内容相同，高度匹配。

演示代码
内容1
内容2
<div class="flex">
  <div class="doc-flex-item">内容1</div>
  <div class="doc-flex-item">内容2</div>
</div>
Class Description
.flex 创建具有块元素特征的 flex 容器。
.flex-inline 创建具有行内元素特征的 flex 容器。

水平对齐

这些类定义了flex元素的水平对齐方式，并在它们之间分配间距。 将其中一个或多个添加到flex容器中来flex元素的对齐方式。 flex元素默认为左对齐，和.flex-left 类效果相同。

演示代码
Item 1
Item 2
Item 3
<div class="flex flex-center">
  <div class="doc-flex-item">Item 1</div>
  <div class="doc-flex-item">Item 2</div>
  <div class="doc-flex-item">Item 3</div>
</div>
Class Description
.flex-left 添加这个class将flex项目左对齐。
.flex-center 添加这个class将flex项目沿主轴居中对齐。
.flex-right 添加这个class将flex项目右对齐。
.flex-between 添加这个class将flex项目以中心主轴为准均匀分配，使元素之间间距相等。
.flex-around 添加这个class可均匀分配元素，使每个 flex 元素两侧具有相等的间距。

垂直对齐

这些类定义了flex项的垂直对齐方式。默认情况下，flex项目像.flex-stretch类一样填充其容器的高度。

Message

全局展示操作反馈信息。

何时使用：

  • 可提供成功、警告和错误等反馈信息。
  • 顶部居中显示并自动消失，是一种不打断用户操作的轻量级提示方式。
演示代码 

<p class="doc-msg-btns">
  <button class="btn btn-primary">common</button>
  <button class="btn btn-success">success</button>
  <button class="btn btn-danger">error</button>
  <button class="btn btn-warning">warning</button>
  <button class="btn btn-info">loading</button>
</p>
// message
$(".doc-msg-btns .btn").click(function () {
  var type = $(this).text();
  message({
    type: type, // 类型
    content: '这是一个关于 <b>' + type + '</b> 类型的消息。', // 内容
  });
});

API

参数 说明 类型 必填 默认
type 信息类别：common、success、success、error、warning、loading string 否 common
content 信息内容 string 是
duration 延时消失时间，秒为单位，默认3秒自动消失，如果设置为0，则永不消失 number 否 3
className 消息框增加一个class string 否
onClose 关闭时执行回调 function 否
